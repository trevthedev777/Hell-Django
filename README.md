### Installation and Packages

- Python 3.8.11
```
python --version
```

- Django 
```
pip3 install 'django<4'
```

- Create a new Django project
```
django-admin startproject django_todo .
```

- Run the project
```
python3 manage.py runserver
```

- Create a app
```
python3 manage.py startapp <appname>
```
